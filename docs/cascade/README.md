# Windsurf Cascade 完全教程

## 概述

Windsurf Cascade 是一个强大的 AI 编程助手，它提供了代码/聊天模式、工具调用、语音输入、检查点、实时感知和 linter 集成等功能。Cascade 通过智能代理技术，实现了人机协作的新水平。

## 核心特性

### 1. 基础功能
- **代码/聊天模式**: Code 模式允许 Cascade 创建和修改代码库，Chat 模式优化了代码库问题和通用编程原则的问答
- **模型选择**: 从多种可用模型中选择，包括 SWE-1.5、swe-grep、SWE-1 等
- **计划和待办事项列表**: 内置规划能力，通过专门的规划代理持续完善长期计划
- **排队消息**: 在 Cascade 完成当前任务时，可以排队新消息按顺序执行

### 2. 高级功能
- **工具调用**: 拥有搜索、分析、网络搜索、MCP 和终端等多种工具
- **语音输入**: 使用语音与 Cascade 交互
- **命名检查点和恢复**: 可以恢复 Cascade 所做的更改
- **实时感知**: 感知您的实时操作，无需提示上下文
- **发送问题到 Cascade**: 将编辑器中的问题直接发送到 Cascade
- **解释和修复**: 高亮错误并点击"解释和修复"让 Cascade 修复

### 3. 企业功能
- **记忆和规则**: 在 Cascade 对话中持久化上下文
- **技能**: 处理复杂的多步骤任务
- **工作流**: 自动化重复性任务
- **MCP 集成**: 模型上下文协议集成
- **Cascade Hooks**: 自定义钩子功能
- **分析和 API**: 使用分析和 API 访问

## 快速开始

要打开 Cascade，请按 `Cmd/Ctrl+L` 或点击 Windsurf 窗口右上角的 Cascade 图标。编辑器或终端中选中的任何文本将自动包含在内。
