# 第一章：入门指南

## 1.1 安装和设置

### 系统要求
- 支持的操作系统：Windows、macOS、Linux
- 支持的 IDE：VS Code、JetBrains 系列、Eclipse、Visual Studio
- 网络连接：需要互联网连接以访问 AI 模型

### 安装步骤
1. 下载并安装 Windsurf 编辑器
2. 启动 Windsurf 并登录账户
3. 完成初始设置向导
4. 选择编辑器主题和配置

### 首次配置
- 导入 VS Code 配置（可选）
- 设置首选项和工作区
- 配置代理设置（如需要）

## 1.2 基础界面

### 打开 Cascade
- **快捷键**: `Cmd/Ctrl+L`
- **鼠标**: 点击右上角的 Cascade 图标
- **自动包含**: 选中的文本会自动包含在对话中

### 界面布局

## 1.3 基本操作

### 发送消息
1. 在输入框中输入问题或指令
2. 按 `Enter` 发送
3. 等待 Cascade 响应

### 代码模式 vs 聊天模式
- **代码模式**: Cascade 可以直接修改您的代码库
- **聊天模式**: 适合问答和一般编程讨论

### 基本命令
- `/help` - 显示帮助信息
- `/clear` - 清除对话历史
- `/model` - 切换 AI 模型

## 1.4 第一个示例

### 创建一个简单的 Python 程序

**用户输入**:
```
创建一个计算斐波那契数列的 Python 函数
```

**Cascade 响应**:
```python
def fibonacci(n):
    """计算斐波那契数列的第 n 项"""
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# 测试函数
for i in range(10):
    print(f"fibonacci({i}) = {fibonacci(i)}")
```

## 1.5 配置选项

### 模型选择
- **Base Model ⚡**: 基础模型，快速响应
- **Windsurf Premier 🚀**: 高级模型，更强大的能力
- **其他模型**: GPT-4o、Claude 3.5 Sonnet

### 设置访问
- 点击设置图标
- 配置首选项
- 调整模型参数
- 设置快捷键

## 1.6 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl+L` | 打开 Cascade |
| `Cmd/Ctrl+Enter` | 发送消息 |
| `Esc` | 取消当前操作 |
| `Tab` | 接受建议 |
| `↑/↓` | 浏览历史消息 |

## 1.7 故障排除

### 常见问题
1. **Cascade 不显示**: 检查网络连接和登录状态
2. **响应缓慢**: 尝试切换到基础模型
3. **无法编辑文件**: 确保在代码模式下

- 查看 [故障排除指南](./06-troubleshooting.md)
- 访问官方文档
- 联系支持团队
