# 学习与笔记方法论收敛

## 一、目标层：明确要解决什么
- 学习目标：理解、记忆、应用、创新？
- 输出目标：考试、写作、项目？

## 二、策略层：选择合适方法
- **理解类**：费曼技巧、第一性原理
- **记忆类**：间隔重复、检索练习
- **结构类**：子弹笔记、康奈尔笔记
- **连接类**：Zettelkasten、思维链/逻辑链
- **控制类**：元认知、认知负荷管理

## 三、工具层：落实到日常流程
- 记录：子弹笔记/康奈尔
- 提炼：卡片盒笔记
- 复习：间隔重复+自测
- 反思：元认知日志

## 四、统一闭环（建议流程）
1. **输入**：阅读/课程
2. **转化**：费曼复述 + 卡片化
3. **连接**：建立知识链
4. **复习**：间隔重复+检索练习
5. **反思**：元认知评估
6. **输出**：写作/讲解/项目

## 五、落地原则
- **少而精**：选2-3个方法长期坚持
- **输出驱动**：每次学习必须有输出
- **周期复盘**：每周检查方法效果

## 总结
不同方法对应不同学习阶段：
- **理解**：费曼技巧、第一性原理
- **记忆**：间隔重复、检索练习
- **结构**：子弹笔记、康奈尔
- **系统化**：Zettelkasten、逻辑链
- **控制**：元认知、认知负荷

最终目标是形成“输入→转化→连接→复习→反思→输出”的学习闭环。
