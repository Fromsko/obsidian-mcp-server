# 检索练习与主动回忆

## 核心观点
**检索练习（Retrieval Practice）**强调“从脑中取出”比“反复阅读”更有效。主动回忆能强化记忆痕迹，暴露知识漏洞，形成更稳固的长期记忆。

## 为什么有效
- **生成效应**：自己提取答案比被动输入更深刻。
- **错误暴露**：回忆失败点=学习重点。
- **迁移能力**：主动回忆更接近真实应用场景。

## 实践方式
1. **闭卷复述**：读完一段立刻用自己的话复述。
2. **自测题库**：写出问题清单，定期自测。
3. **空白纸法**：不看资料写出知识结构。
4. **口头讲解**：模拟讲解给他人。

## 注意事项
- 回忆失败不要焦虑，它是定位薄弱点的信号。
- 回忆后要立刻校对，修正错误。

## 与其他方法结合
- 与**费曼技巧**结合：回忆+讲解双重强化。
- 与**间隔重复**结合：在最易遗忘时做回忆。

## 小结
检索练习的本质是“用输出驱动输入”，是学习效率提升的第一抓手。
