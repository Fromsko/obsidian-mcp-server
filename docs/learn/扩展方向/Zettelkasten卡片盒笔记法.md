# Zettelkasten 卡片盒笔记法

## 核心理念
Zettelkasten 强调“原子化笔记 + 链接形成网络”，让知识在笔记系统中自我生长。

## 关键原则
1. **原子化**：一张卡片只写一个观点
2. **用自己的话**：避免照搬原文
3. **双向链接**：让概念之间形成网络
4. **持续迭代**：旧笔记不断被新笔记激活

## 典型流程
- 读书/学习 → 写临时笔记
- 提炼为永久笔记
- 建立连接与索引

## 实践技巧
- 每条笔记写“能独立理解”的内容
- 建议标题=结论句
- 建立“主题索引卡”统筹大方向

## 适用场景
- 长期知识积累
- 写作与研究
- 概念体系构建

## 小结
卡片盒法本质是“让知识彼此对话”，适合深度研究者。
