# 深度阅读与 SQ3R 笔记法

## 深度阅读的核心
深度阅读强调“理解结构 + 主动加工 + 多轮回顾”。它不是速度阅读，而是**为了理解、迁移和输出而读**。

## SQ3R 方法概览
SQ3R 是经典的深度阅读框架：
1. **Survey（浏览）**：快速扫读目录、标题、摘要
2. **Question（提问）**：把标题转成问题
3. **Read（精读）**：带着问题阅读
4. **Recite（复述）**：合上书，用自己的话复述
5. **Review（复习）**：隔天/隔周回顾

## 实操流程（可直接照做）

### 1. 浏览（Survey）
- 先看目录与小标题
- 标记核心章节
- 判断是否值得深读

### 2. 提问（Question）
- 把每个小标题写成问题
- 例如“什么是X？”“为什么Y会失败？”

### 3. 精读（Read）
- 带着问题逐段阅读
- 仅标记“关键结论”和“关键依据”

### 4. 复述（Recite）
- 合上书，写 5-10 句摘要
- 或用费曼方式讲给自己听

### 5. 复习（Review）
- 24小时内回顾一次
- 1周后再回顾一次
- 搭配间隔重复效果更强

## 深度阅读的标注体系

### 推荐标记规则
- **⭐ 结论**：最重要的核心观点
- **❓疑问**：看不懂或需要查证
- **🔗链接**：能连接到已有笔记的点
- **🧠启发**：个人联想或应用场景

### 标注原则
- 不要大段高亮
- 每页最多标注3-5点
- 标注要能“独立复述”

## 输出型笔记模板

```
【标题】
- 核心结论：
- 关键论据：
- 重要概念：
- 与已有知识的连接：
- 个人应用/启发：
```

## 常见误区

1. **只划线不复述**：没有输出=没有真正理解
2. **边读边做大纲**：破坏注意力，建议先读后整理
3. **追求速度**：深读的目标是“可迁移”

## 与其他方法结合

- **费曼技巧**：用复述检测理解
- **卡片盒**：输出为原子笔记
- **检索练习**：复习阶段做自测

## 小结
深度阅读的关键在于**问题驱动 + 主动复述 + 多轮回顾**。SQ3R 是一套成熟的流程，配合标注体系与笔记模板，就能把阅读真正变成可输出的知识积累。
