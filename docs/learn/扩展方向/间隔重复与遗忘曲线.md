# 间隔重复与遗忘曲线

## 核心观点
**间隔重复（Spaced Repetition）**通过在遗忘临界点复习，最大化记忆巩固效率。

## 遗忘曲线要点
- 记忆随时间快速衰减
- 在即将遗忘时复习，强化效果最强

## 实践方案
1. **复习节奏**：1天、3天、7天、14天、30天……
2. **错题系统**：把错误内容纳入复习节奏
3. **卡片化**：将知识拆成最小单元

## 工具建议
- **Anki**：算法支持的复习卡片
- **RemNote**：笔记与复习结合
- **Obsidian插件**：Spaced Repetition

## 常见误区
- 复习过密导致疲劳
- 卡片内容过大

## 小结
间隔重复=“在遗忘边缘复习”，是长期记忆的工程化路径。
