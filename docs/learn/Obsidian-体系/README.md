# Obsidian 学习体系（面向：体系化/梳理/深度/本质/扩展/启迪）

## 目标对齐
- **建立体系**：用索引+双向链接形成知识网络
- **梳理知识**：用结构化模板沉淀
- **深度学习**：用问题驱动 + 费曼复述
- **直击本质**：用第一性原理与因果拆解
- **扩展思路**：用对比/交叉/迁移视角
- **启迪思考**：用灵感卡与行动卡收敛输出

## 目录建议（Obsidian 格式）
```
learn/
  00-索引/
  10-概念卡/
  20-读书/课程笔记/
  30-问题链/
  40-本质拆解/
  50-扩展思路/
  60-灵感启发/
  70-复盘与行动/
  Templates/
```

## 核心工作流（输入→转化→连接→输出）
1. **输入**：读书/课程 → 20-读书/课程笔记
2. **转化**：提炼概念 → 10-概念卡
3. **连接**：建立逻辑链 → 30-问题链
4. **本质**：因果拆解 → 40-本质拆解
5. **扩展**：交错/迁移 → 50-扩展思路
6. **启发**：输出行动 → 60-灵感启发 + 70-复盘与行动

## 使用建议
- 每次阅读至少产出 1 张「概念卡」+ 1 条「问题链」
- 每周形成 1 次「本质拆解」或「复盘行动」
- 每条笔记必须能独立理解
